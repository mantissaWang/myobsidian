# Text Elements
```js
import React from "react";
import ReactDOM from "react-dom";


function Hello(props) {
        return <h1>Hello, {props.name} <h1>
}

const rootElement = document.getElementById("root");
ReactDOM.render(<Hello name="heidi"/>, rootElement);
``` ^8w4Z9Q6I

這個Hello大寫的function，
是用React Component宣告的，
所以叫做React Component ^lVFXjvsW

這個是JSX描述的React DOM elements結構，
叫做 React Elements。 ^nMMMA8Z9

渲染React Elements時，
背後會建立React Component instances。 ^95YuGTWW

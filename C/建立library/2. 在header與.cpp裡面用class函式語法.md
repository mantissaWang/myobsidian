# 1、class是甚麼？
通常class會寫在header檔案裏面。
class裡面會包含public、private。

## (1) 關於public？
寫在public裡面的程式碼，你可以從其他的program來呼叫。

## (2) 關於private？
寫在private的程式碼，你可以在CPP裡面使用，但是不能在main scketch裡面用。
